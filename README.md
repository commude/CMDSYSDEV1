## Configuration
1.  Clone this repository.
2.  Recreate and configure `.env` by copying `.env.example`
    ```bash
        $   cp .env.example .env
    ```
3.  Install composer
    ```bash
        $   composer install
    ```
4.  Generate Application Key
    ```bash
        $   php artisan key:generate
    ```
